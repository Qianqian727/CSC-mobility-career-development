# CSC-mobility-career-development

This repository contains the code used for the analysis of academic career development associated with different China Scholarship Council (CSC) international mobility schemes.

## Repository contents

- `01_Career progression-KM survivial analysis.ipynb`  
  Kaplan–Meier survival analysis for career progression.

- `02_Scholarly performance.ipynb`  
  Analysis of scholarly performance indicators.

- `03_Academic embeddededdness.ipynb`  
  Analysis of academic embeddedness patterns.

## Data availability

The full research dataset is archived on Zenodo：https://doi.org/10.5281/zenodo.19601444 

This GitHub repository contains code only.  
No full research data are provided here.
